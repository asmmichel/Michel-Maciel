Código Lista SEO

A lista Simplismente Encadeada e Ordenada (SEO) usa em seu código ponteiros de inicio, fim, aux e anterior, junto com os elementos func e prox, esses ponteiros trabalham entre si para limpar ou adicionar novos dados em ordem alfabética, comparando os objetos adicionados.

O código foi feito através da estrutura de dados de lista, as opções do código são inserir, listar, remover e apagar todos os funcionários.

Código Fila de Estacionamento

O código simula um pátio de veículos de um pequeno apartamento ou conjunto, onde é possível estacionar apenas 5 carros no total, todos os carros são permitidos a entrar apenas por uma posição e sair apenas por outra, ou seja, se o pátio está lotado e o terceiro carro deseja sair, o primeiro e o segundo terão que sair e entrar novamente para dar espaço para a saída do terceiro.

Isso ocorre porque a estrutura de dados usada no código é a de Fila, aonde o sistema de controle é do tipo First In, Last Out (FILO), as opções do código são de entrada, saída, listagem e esvaziar veículos do pátio.

Código Torre de Pilhagem

Código que demonstra uma torre de números aonde o primeiro começa na base e o último sempre está no topo, a estrutura de dados usada no código é a de pilha, aonde o sistema de controle é do tipo First In, First Out (FIFO), por isso que só é possível retirar do topo os números colocados.

Diferente da Lista e da Fila, não é possivel escolher qual dado retirar nesse código, sendo obrigatório sempre a saída do número ao topo, as opções do código são inserir, retirar, mostrar, e apagar os números.

Código Lista DEO

A lista Duplamente Encadeada e Ordenada (DEO) usa um sistema de controle parecido com a SEO, a diferença aqui é que a lista SEO tinha apenas um ponteiro prox em cada elemento, sendo possível apontar apenas o que estava abaixo dela, ja a lista DEO tem um ponteiro para cima e para baixo dela, usando o "ant" agora junto como um Elemento, sendo então "Duplamente Encadeada".

O código foi feito através da estrutura de dados de lista, as opções do código são inserir, mostrar, remover e apagar todos os números.

Código Somatório Inverso

Significado de somatório inverso: 1-2+3-4+5-6 ate +-n, aonde n é o valor final até ser somado que o usuário definiu. O código usa recursividade, aonde uma função chama a si mesma para executar o somatório, os argumentos (n - 1, -nUM) que fazem ser possível a troca de sinais + e - a cada vez que se chama a função recursiva.

Código Lista de Filmes

Este código não é um código utilizável e completo, apenas uma demonstração de manuseamento de objetos e variáveis, o código ira pedir o Nome e Gênero de um filme, como está usando DO-While, só vai ser parada a chamada caso a Flag String N/n seja chamada, após isso, irá ser mostrada a listagem dos filmes e a listagem ORDENADA dos filmes.
